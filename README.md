# 職務要約
これまで4年間に渡り、モバイルアプリエンジニアとしてシステム開発に従事しています。

1社目では、会員登録者数160万人を超える、大規模LIVE配信プラットフォーム「SHOWROOM」のiOSエンジニアとして従事してまいりました。
iOSチームのリードエンジニアを務め、各メンバーの担当プロジェクトのマネジメントや新技術の導入検討を主導するなどをしておりました。
担当プロジェクトとしては課金基盤のリプレイスや大規模改修プロジェクトのiOS主担当を務めるなど、サービスの品質向上・開発環境の改善に貢献しました。

現在は、LINEヤフー株式会社にて、ユーザ数約1000万人に登るフリマアプリ・サービス「Yahoo!フリマ」のiOSエンジニアとして従事しております。
入社後、すぐにE2E自動テストの導入基盤の構築などを主導し、チームの業務効率向上に大きく貢献しました。
現在ではAI開発をメインに開発業務を行っております。

また、2024年より副業として株式会社any styleにてVTuber とユーザーのコミュニケーションを主軸とした推し活アプリ「my dear.」のモバイルアプリエンジニアとして従事しており、iOS/Androidアプリの開発業務に従事しております。
