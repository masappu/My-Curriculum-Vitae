これまで4年間に渡り、モバイルアプリエンジニアとしてシステム開発に従事しています。

1社目では、会員登録者数160万人を超える、大規模LIVE配信プラットフォーム「SHOWROOM」のiOSエンジニアとして従事してまいりました。
iOSチームのリードエンジニアを務め、各メンバーの担当プロジェクトのマネジメントや新技術の導入検討などをしておりました。

現在は、LINEヤフー株式会社にて、ユーザ数約1000万人に登るフリマアプリ・サービス「Yahoo!フリマ」のiOSエンジニアとして従事しております。

また、2024年より副業として株式会社any styleにてVTuber とユーザーのコミュニケーションを主軸とした推し活アプリ「my dear.」のiOS/Androidアプリの開発業務に従事しております。

# 職務経歴
## LINEヤフー株式会社
ユーザ数約1000万人に登るフリマアプリ・サービス「Yahoo!フリマ」のiOSエンジニアとして従事しております。<br>
入社後は特に、E2E自動テストの導入基盤構築を主導し、テスト設計からCI/CD連携までを整備するとともに、チーム内向けに自動テスト実装を支援するMCPツールを自ら設計・実装し展開しました。<br>
これにより、属人化していたE2Eテスト実装を標準化し、リグレッション検知の自動化・レビュー工数の削減・リリース前品質の安定化を実現し、チーム全体の開発効率と品質水準を大きく底上げする成果を創出しました。<br>
現在は通常のiOS開発と並行して、AIの導入に注力しており、Claude Code・Codex・GitHub Copilotなどを活用したAI駆動開発フローを整備しています。<br>
設計レビュー、テストコード生成、リファクタ支援などにAIを組み込むことで、開発スピードの向上と品質担保を両立する持続的な開発体制の構築に貢献しています。<br>
